Demo page for instant-vlist

This page renders auto-generated NDJSON stream of book-like records. It demonstrates how instant-vlist can processes records streams of varying quantity and transfer speed.

Use the search box and facet filters to narrow the stream, choose how many records to fetch with Records To Fetch, adjust Delay to change the server-side streaming pause between sending each record, and jump directly to a record number with Jump To Record.

NOTE: This demo uses free hosting and the server may not always return the desired amount of records if it is too big
