Demo page for instant-vlist

This page renders auto-generated NDJSON stream of book-like records. It demonstrates how instant-vlist can processes streams of records of different quantity of records and at different transfer speed.

Use the search box and facet filters to narrow the stream. For Category selection use Ctrl+Clik for multiple choices. Choose how many records to fetch with Records To Fetch, adjust Delay (not for 1M records though) to change the server-side streaming pause before sending next record (may be useful for checking ), and jump directly to a record number with Jump To Record.

NOTE: This demo was written using AI and it uses free hosting. The server may not always return the desired amount of records if used extensively. 1 Million records is streamed from pre-generated file
